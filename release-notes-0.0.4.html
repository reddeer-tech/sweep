What's New

🔎 Search in Optimize

The Memory and CPU screens now have a search box. Type any part of an app name or its vendor / Team ID to instantly filter the list down to what you're looking for — handy when 600+ apps are running. The count updates to show matches (e.g. "3 of 612 apps"), and clearing the box brings everything back. Search applies to whichever tab you're on.

Under the hood

  • make deploy now accepts release notes directly: make deploy NOTES='…' (Markdown) or make deploy NOTES_FILE=notes.md — the text is converted to HTML and embedded in the update feed automatically.
